Этот парсер создан для переноса сайта с одной CMS на другую (WordPress). 
Он учитывает особенности плагина для импорта на WordPress'е. Благодаря этому 
имеет структуру выходного файла с теми полями, которые указал заказчик и всей
необходимой информацией: Url, текст, изображения, хлебные крошки, цены, характеристики товара.
В необходимых местах остаются элементы верстки (заголовки, параграфы и тд),
также забираются некоторые мета-теги для seo-продвижения. 


Пауки для каждого уровня вложенности находятся в разных файлах.

В репозиторий проекта прикрепляю пример полученых файлов в папку Result
(парсер их отдает в формате  csv - я их преобразовала в excell для вашего удобства).
